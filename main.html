About TruthCheck

🎯
Mission

Combat misinformation using AI-powered linguistic analysis

TruthCheck was built to address the growing problem of fake news and misinformation spreading through social media and messaging platforms. Our mission is to provide users with an instant, AI-powered tool to analyze suspicious content before sharing it. We focus on linguistic patterns, sentiment analysis, and credibility indicators to help users make informed decisions about the information they consume and share.

NLP Engine

TextBlob for sentiment analysis & linguistic patterns

TextBlob Library: Built on top of NLTK, provides simple API for common NLP tasks

Capabilities:
  • Sentiment Analysis: Polarity (-1 to +1) and subjectivity (0 to 1) scores
  • Part-of-Speech Tagging: Identifies nouns, verbs, adjectives
  • Tokenization: Breaks text into words and sentences
  • Word Frequency: Analyzes lexical diversity
NLTK Corpora Used: Brown corpus, Punkt tokenizer, WordNet, Averaged Perceptron Tagger, ConLL2000, Movie Reviews

Analysis Features

Sentiment polarity, subjectivity detection, clickbait patterns, credibility indicators

Manipulation Detection:
  • Clickbait phrase detection (e.g., "you won't believe", "shocking")
  • Urgent/fear language patterns
  • Excessive capitalization analysis
  • Punctuation abuse detection (!!!, ???)
  • Emotional manipulation words
  • Low lexical diversity (repetitive language)
Credibility Signals:
  • Source citations (e.g., "according to", "study shows")
  • URL presence for verification
  • Specific dates and numbers
  • Statistical data references
  • Neutral, objective tone
  • Rich vocabulary and diverse language

Algorithm

Multi-factor scoring with sigmoid probability distribution

Scoring System:
  • Credibility Score: Positive signals (sources, URLs, data, neutral tone)
  • Manipulation Score: Negative signals (clickbait, urgency, bias)
  • Net Score: Manipulation - Credibility
Sigmoid Function:
probability = 1 / (1 + e^(-score * 0.6))

Why Sigmoid? Provides smooth probability curve, prevents extreme 0% or 100% scores, handles outliers gracefully

Thresholds:
  • ≥62%: Classified as "Fake"
  • ≤38%: Classified as "Real"
  • 38-62%: Classified as "Uncertain"
Confidence Calculation: For "Real" labels, confidence = (1 - fake_probability) × 100. For "Fake" labels, confidence = fake_probability × 100. Capped at 55-95% range.
